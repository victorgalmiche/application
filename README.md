# Probabilité de survie sur le Titanic

Pour pouvoir utiliser ce projet, il 
est recommandé de créer un fichier `.env`
ayant la structure suivante:

```yaml
JETON_API="####" #renseigner la valeur pertinente
```

Le déploiement de l'application est contrôlé par le dépôt application-deployment. 
Elle est disponible à l'URL https://victor-titanic.lab.sspcloud.fr
